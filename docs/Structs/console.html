  • log(_:condition:separator:terminator:className:lineNumber:)

    Writes the textual representations of debug message, separated by separator and terminated by terminator, into the standard output.

    Declaration

    Swift

    public static func log(_ messages: Any..., condition: Bool = true, separator: String = " ", terminator: String = "\n", className: String = #file, lineNumber: Int = #line)

    Parameters

    messages

    Messages to write to standard output.
    condition

    To achieve assert like behavior, you can pass condition that must be met to write ouput.
    separator

    The separator to use between messages. The default value is " ".
    terminator

    To print without a trailing newline, pass terminator: "". The default value is "\n".
    className

    The name of the class where this log is executed. The default value is extracted from #file.
    lineNumber

    The line number where this log is executed. The default value is of #line.
